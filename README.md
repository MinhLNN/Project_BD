## Using LSTM Model to predict the number of casualties in the UK with PySpark

Traffic accidents are one of the leading causes of death and injury worldwide. According to the World Health Organization (WHO), traffic accidents claim approximately 1.35 million lives annually and cause over 50 million serious injuries.
With the rapid advancement of technology and Big Data, the application of data analysis and machine learning techniques has emerged as a promising solution for predicting and preventing traffic accidents.
In this research, we conduct an in-depth analysis using Long Short-Term Memory (LSTM) to predict the number of casualties.

### Built With

* PySpark
